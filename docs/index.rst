StochSS-Compute |release|
#########################

StochSS-Compute is a compute delegation server for the `StochSS <https://github.com/StochSS>`_ family of stochastic simulation software. StochSS-Compute allows for one to run StochSS or GillesPy2 simulations on distributed cloud compute resources.

Getting a copy of GillesPy2
***************************

The latest version of GillesPy2 can be found on `PyPI <https://pypi.org/project/gillespy2>`_.  The source code is available on `GitHub <https://github.com/StochSS/GillesPy2>`_.  GillesPy2 is licensed under the GNU General Public License version 3.

Getting help
************

If you find any problem with StochSS-Compute or this documentation, please report it using `the GitHub issue tracker <https://github.com/StochSS/stochss-compute/issues>`_ for the project.  You can also contact the main author, `Matthew Dippel <https://github.com/mdip226>`_, directly with questions and suggestions.
